O que é chave ssh:

-Modo de autenticação mais segura do Git para o Github,
- Uma forma de estabelecer uma coneção mais segura e encriptada entre duas maquinas.
 
	 fazer uma chave ssh:
		.ssh-keygen -t ed25519 -c **e-mail** 
		.cd /c/user/---/.ssh/
		.cat  id25519 .pub (+chave ssh do github)
